# Exercícios de JAVA

Este repositório reúne os exercícios, desafios e projetos desenvolvidos durante meus estudos em JAVA.

O objetivo é registrar minha evolução na linguagem, praticar lógica de programação e aplicar os conceitos aprendidos ao longo dos cursos e estudos.

---

## Objetivos

- Praticar lógica de programação.
- Aprender os principais recursos da linguagem JAVA.
- Resolver desafios propostos durante os estudos.
- Registrar minha evolução como desenvolvedor.
- Criar uma base de consulta para projetos futuros.

---

## Conteúdo

Os exercícios abordam diversos assuntos, como:

- Variáveis e tipos de dados
- Operadores
- Entrada e saída de dados
- Estruturas condicionais
- Estruturas de repetição
- Funções (Métodos)
- Vetores (Arrays)
- Matrizes
- Listas
- Programação Orientada a Objetos (POO)
- Construtores
- Encapsulamento
- Sobrecarga
- Herança
- Polimorfismo
- Enumerações
- Composição
- Tratamento de exceções
- Manipulação de arquivos
- Interfaces
- Classes abstratas
- Generics
- Coleções
- Lambda e Stream API
- JDBC
- JavaFX (quando aplicável)
- Exercícios práticos

---

## Organização

Os arquivos estão organizados de acordo com a origem dos exercícios (faculdade ou curso). Cada tema possui sua própria pasta e, dentro dela, os exercícios são organizados em diretórios como `ex00`, `ex01`, etc., facilitando a localização dos arquivos.

Exemplo:

```text
📂 Exercicios-OOP
│
├── ex00
├── ex01
├── ex02
├── ex03
├── ex04
└── README.md
```
